# poor man's pong

A simple Pong game for me to learn the very basics of C and SDL.

The code is ugly, I know. But I don't see a pong game in your git repo?

If (for some reason) you fancy giving this a try, clone the repository, and run:

```bash
make build;
./game;
```

To control the game, it's `D` & `F` to control the left paddle, and `J` & `K` to control the right one, because i'm not smart enough to write an opponent ai.
